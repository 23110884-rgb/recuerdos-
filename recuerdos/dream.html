Dreamweaver

En Adobe Dreamweaver se aprende a diseñar, programar y publicar sitios web y aplicaciones web adaptables . A diferencia de Photoshop, no trabaja con imágenes individuales, sino con la estructura, estética y funcionalidad de páginas de internet completas utilizando código y herramientas visuales.

Lenguajes de programación y estructura web

  • Estructuración con HTML: Organizar el contenido de una web (textos, títulos, imágenes, enlaces y tablas) usando etiquetas estándar.

  • Estilizado con CSS: Controlar el diseño visual, incluyendo colores, fuentes, espaciados y la disposición de los elementos.

  • Interactividad con JavaScript: Añadir funciones dinámicas, menús interactivos, validación de formularios y animaciones básicas.

  • Conexión a bases de datos: Integrar lenguajes de servidor como PHP para gestionar sitios dinámicos o sistemas de inicio de sesión. 

Modos de trabajo y flujos de maquetación

  • Uso de tres vistas clave: Alternar entre la vista Código (programación pura), la vista Diseño (visual tipo arrastrar y soltar) y la vista Dividida (ambas en tiempo real).

  • Maquetación adaptable: Crear cuadrículas fluidas y usar Media Queries para que las páginas funcionen perfectamente en celulares, tabletas y computadoras.

  • Gestión de plantillas: Construir bloques de código reutilizables para aplicar cambios globales a cientos de páginas al mismo tiempo. 

Administración y publicación de proyectos

  • Arquitectura de archivos: Organizar carpetas locales, hojas de estilo externas, scripts e imágenes para mantener un flujo limpio.

  • Conexión por FTP: Subir los archivos directamente desde el programa a un servidor de alojamiento web (hosting) en la nube.

  • Depuración de errores: Utilizar el motor de sugerencias de código y previsualización en vivo en diferentes navegadores para corregir fallos técnicos.
